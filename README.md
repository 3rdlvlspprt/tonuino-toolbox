## Development

Um am Projekt mit zu wirken klone das Repository, auf Deinem rechner muss git und npm eingerichtet sein.

```bash
# Repo klonen
git clone https://github.com/raph-ael/tonuino-toolkit.git

# ins Verzeichnis wechseln
cd tonuino-toolkit

# Pakete installieren
npm install

# dev Version starten
npm start
```
